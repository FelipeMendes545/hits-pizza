# 🍕 Hit's Pizza - Menu Digital & Sistema de Delivery

O *Hit's Pizza* é uma plataforma web completa de delivery e menu digital desenvolvida para otimizar e digitalizar o processo de vendas e atendimento de uma pizzaria. O projeto deixou de ser apenas um protótipo acadêmico para se tornar uma solução real e funcional aplicável ao mercado de food service.

A aplicação conta com um backend robusto construído em Python, uma interface totalmente responsiva e uma lógica fluida para a navegação de usuários e fechamento de pedidos.

---

## 🚀 Funcionalidades Principais

* *Menu Interativo:* Exibição dinâmica dos produtos com detalhes, preços e categorias.
* *Carrinho de Compras Dinâmico:* Adição, remoção e atualização de itens em tempo real antes da finalização.
* *Interface Responsiva:* Experiência de usuário otimizada tanto para desktops quanto para dispositivos móveis (smartphones e tablets).
* *Processamento de Pedidos:* Integração da lógica de backend para estruturação, validação e fechamento do pedido do cliente.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias e boas práticas de mercado:

* *Backend:* [Python](https://www.python.org/) com o micro-framework [Flask](https://flask.palletsprojects.com/) (gerenciamento de rotas, renderização de templates e lógica de negócios).
* *Frontend:* HTML5 para estruturação semântica e CSS3 para estilização moderna e layouts responsivos.
* *Controle de Versão:* Git e GitHub para gerenciamento de código.
